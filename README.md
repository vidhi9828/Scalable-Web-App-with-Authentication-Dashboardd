# Project Overview

A simple and responsive landing page built using HTML, CSS, and JavaScript.  
It features a smooth UI, clean layout, and essential sections like Hero, Features, About, and Contact.  
The project is lightweight, fast, and ideal as a beginner-friendly web development assignment.

## Features
- Fully responsive UI  
- Clean and modern design  
- Fast loading and minimal code  
- Easy to customize  

## Tech Stack
- HTML5  
- CSS3  
- JavaScript

## How to Use
1. Clone or download the project.  
2. Open `index.html` in your browser.  
3. Customize as needed and extend it further.
